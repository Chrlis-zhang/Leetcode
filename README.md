## 简介

这个项目里，存放的是我用**Python**编写的题目答案。我将所有答案的**运行时间**／**排名比例**都粘贴在了答案的上方，便于你快速浏览与思考这个答案存在的**缺点**／**优点**。当你阅读完代码存在困惑时，可以查看对应的Commit里的提示。


## 使用说明

直接用Ctrl + F，通过相应的题目名称去搜索即可。
